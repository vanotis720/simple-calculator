## Run Locally

Clone the project

```bash
  git clone https://github.com/vanotis720/simple-calculator.git
```

Go to the project directory

```bash
  cd simple-calculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  expo start
```
